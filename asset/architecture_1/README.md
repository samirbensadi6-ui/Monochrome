## Projet landing page : consignes

Bienvenue dans cet exercice de création de portfolio qui vous permettra d'appliquer et de consolider les concepts que vous avez appris jusqu'à présent en conception web.
Objectif de l'exercice

Vous allez **créer une page web, qui sera la page d’accueil du site vitrine d’un professionnel.**
Ce dernier vous fournit les données à intégrer, et des références pour cadrer le projet :

  Une ébauche de **charte graphique**
  Une **maquette basse fidélité** correspondant à ses attentes.

**Les images fournies ont été soit récupérées sur des sites d’images libres de droits, soit générées par IA.**

Les recommandations graphiques sont une base de travail pour éviter la feuille blanche, et peuvent être challengées si vous avez des suggestions d’UI ou de visuels plus appropriés.

Pour cet exercice, **vous pouvez choisir parmi plusieurs jeux de données,** disponibles sur la page suivante : 
 -  Les données : https://la-toile-emlyon.github.io/repertoire-donnees-de-sites/static-portfolio-data.html
  - Wireframe : https://la-toile-emlyon.github.io/repertoire-donnees-de-sites/assets/Wireframe%20Landing%20Page.pdf


###Structure sur papier

**Commencez par faire un découpage de la maquette sur papier**, et décomposez ses différentes parties en indiquant à quels éléments HTML chaque partie correspond.

Vous aurez une vue d'ensemble du projet avant d'entamer le travail de code.
Réalisation

    Utilisez l'éditeur de texte VS Code pour écrire votre code HTML et CSS.
    Utilisez GitHub Desktop pour versionner votre code et le publier sur GitHub.
    Utilisez GitHub Pages pour déployer votre site en ligne.
